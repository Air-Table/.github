# Air Table – Flexible Database & Collaboration Platform

## About Air Table

**Air Table** combines the simplicity of a spreadsheet with the power of a relational database, creating a unique platform for modern data management.  
With **Air Table**, teams can organize complex workflows, track projects, manage inventories, and build custom applications without writing code.  
**Air Table** provides an intuitive grid interface, but beneath the surface lies a robust relational database capable of linking records across multiple tables, creating rich data structures that adapt to any business need.  

Whether you are coordinating a marketing campaign, building a product roadmap, or managing a creative production pipeline, **Air Table** offers the flexibility to design your workspace exactly how you need it.  
**Air Table** supports real-time collaboration, allowing team members to comment, assign tasks, and share views instantly.  

For developers and advanced users, **Air Table** includes a comprehensive API and scripting capabilities, enabling deep integration with external services and automation of repetitive tasks.  
**Air Table** also features extensive view options, including Grid, Kanban, Gallery, Calendar, and Gantt views, ensuring that your data is always presented in the most meaningful way.

## Key Features of Air Table

- **Relational Database Architecture**  
  **Air Table** allows you to link records between tables, creating a seamless relational structure that scales with your projects. With **Air Table**, you can manage interconnected data without complex SQL queries.

- **Multiple View Types**  
  **Air Table** provides Grid, Kanban, Gallery, Calendar, and Gantt views. Each **Air Table** view can be filtered, sorted, and grouped independently, offering complete flexibility for different team roles.

- **Real-Time Collaboration**  
  **Air Table** enables synchronous editing, inline comments, and activity logs. Teams using **Air Table** can work together from anywhere with full visibility into changes and discussions.

- **Automation & Scripting**  
  Automate repetitive tasks with **Air Table** automation triggers and actions. **Air Table** supports custom JavaScript scripting for advanced logic and data processing.

- **Rich Field Types**  
  **Air Table** includes attachments, checkboxes, barcode scanning, rollup fields, and lookup fields. These **Air Table** field types allow for sophisticated data modeling directly in the interface.

- **API & Integrations**  
  Connect **Air Table** to thousands of apps via Zapier, Make, or the native **Air Table** REST API. **Air Table** integration capabilities make it a central hub for business data.

- **Interface Designer**  
  Create custom **Air Table** interfaces with drag-and-drop components. **Air Table** interfaces allow non-technical users to interact with data through tailored dashboards.

- **Enterprise-Grade Security**  
  **Air Table** offers SOC 2 compliance, enterprise encryption, and granular permission controls. **Air Table** ensures that sensitive data remains protected across all workspaces.
